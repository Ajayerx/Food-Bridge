What is FoodBridge

ONE PLATFORM.
Every role.
EVERY ORDER.

"FoodBridge is not just a delivery app — it's a complete restaurant management and food delivery ecosystem designed to serve every actor in the chain simultaneously."

Built on a modular monolith architecture with a single Node.js REST API, FoodBridge powers food delivery, dine-in ordering, takeaway management, kitchen coordination, and full platform analytics — all from one unified backend.

React Native · iOS & Android Unified REST API Socket.IO Real-Time Razorpay Payments JWT + OTP Auth Docker · AWS · CI/CD PostgreSQL + Redis
How It Works

EVERY ROLE,
a perfect EXPERIENCE

FoodBridge orchestrates every actor in the food delivery chain through real-time, event-driven workflows — from first tap to delivered order.

🔍

Browse & Discover

Search restaurants by cuisine, view full menus with variants and modifiers, read verified reviews, and find the perfect meal.

🛒

Build Your Cart

Add items with custom variants, choose a delivery address via Google Maps, and apply a coupon code to save on your order.

💳

Pay Securely

Complete payment via UPI, card, or wallet using the embedded Razorpay SDK. HMAC-verified webhooks confirm payment instantly.

🎉

Track & Enjoy

Get live push notifications at every step — accepted, preparing, on the way, and delivered. Then rate your experience.

🏪

Onboard & Go Live

Submit your restaurant profile for admin review. Once approved, configure menus, tables, hours, delivery zones, and assign staff.

📋

Receive Orders Live

New orders appear instantly on your real-time board via Socket.IO — no polling. Accept or reject in one tap with automatic customer notification.

👨‍🍳

Coordinate Kitchen

Kitchen staff tracks every order on the live kitchen display. Assign a delivery agent once food is packed and ready for pickup.

📊

Grow with Data

Review sales reports, monitor commission deductions, respond to customer reviews, and export CSV data to drive informed decisions.

🟢

Go Online

Toggle your availability in the agent app. Your status syncs to Redis and the backend in real time via the heartbeat API.

📲

Receive Task Push

A high-priority FCM/APNs notification wakes the app instantly with full pickup details when a vendor assigns you a task.

🗺️

Navigate to Pickup

Deep-link directly into Google Maps or Apple Maps for turn-by-turn directions to the restaurant. Confirm pickup with one tap.

✅

Deliver & Complete

Mark the order as Delivered. The customer is notified instantly, commission is auto-calculated, and your task queue clears.

🏛️

Platform Oversight

Super Admin has full access to all restaurants, users, orders, and reports. Approve or reject vendor onboarding with a single action.

⚙️

Configure Platform

Set per-restaurant commission rates, manage platform feature flags, configure banners, and issue global coupons from the admin panel.

🎫

Resolve Support

Full access to all support tickets. Thread messages with customers, flag refunds, escalate issues, and close tickets with notifications.

📉

Analyse & Export

View platform-wide sales, vendor payout, and order analytics via Recharts dashboards. Export commission data to CSV for processing.

Core Features

BUILT FOR EVERY
part of THE CHAIN

🛒

Multi-Type Ordering

Online delivery, dine-in table orders, and takeaway — all flowing through the same unified order lifecycle in a single API.

→ Delivery · Dine-In · Takeaway
💳

Razorpay Payments

UPI, cards, wallets, and net banking via Razorpay SDK. HMAC webhook verification, idempotency keys, and automatic refund flow on cancellation.

→ Secure · Idempotent · Verified
🎟️

Smart Coupon Engine

Vendor-specific and platform-wide coupon codes with applicability validation, per-user usage limits, and full redemption tracking.

→ Smart · Trackable · Flexible
⭐

Reviews & Ratings

Post-delivery customer reviews with vendor reply capability, admin-level moderation, and automatic per-restaurant rating aggregation.

→ Transparent · Moderated
📍

Address & Maps

Google Maps SDK-powered address selection, multi-address storage per customer, and agent GPS polling for customer-visible live tracking.

→ Accurate · Multi-Address
🆘

Support Tickets

Full support ticket lifecycle with threaded messaging, status management, and refund-flagging for escalations — visible to customer and admin.

→ Threaded · Escalatable
🍽️

Menu Management

Full category, item, variant, and modifier group management. Upload item photos to S3/CDN, toggle availability instantly, filter by cuisine.

→ Variants · Modifiers · Images
📊

Real-Time Order Board

Live order board powered by Socket.IO — new orders appear the instant they're placed. Accept, reject, and update in real time, no refresh.

→ Socket.IO · Zero-Latency
👨‍🍳

Kitchen Display System

Dedicated kitchen view that streams live order status events via WebSocket — tablet-optimized for kitchen staff to track every dish.

→ Tablet-Optimized · Live
🪑

Table Management

Create tables, track occupancy, assign waiters, and manage full dine-in sessions. Tables auto-release on order completion.

→ Dine-In · Auto-Release
👥

Staff & Role Management

Add managers, waiters, kitchen staff, and cashiers with granular RBAC. Each role sees only the views and actions relevant to their function.

→ RBAC · Granular Access
📈

Sales Reports & Analytics

Pre-aggregated sales, order counts, and delivery performance reports. CSV export for accounting. Commission records auto-calculated per order.

→ Exportable · Auto-Calculated
🔐

OTP Authentication

Passwordless OTP login via SMS for all user types. JWT access tokens (15–60 min) + 30-day refresh tokens with secure storage and forced-logout support.

→ Passwordless · Stateless JWT
🛵

Delivery Management

Vendor-controlled agent assignment, full task lifecycle (Assigned → Picked Up → Delivered), reassignment logic, and real-time push on every event.

→ Full Lifecycle · Real-Time
🔔

Multi-Channel Notifications

FCM, APNs, SMS, and email dispatched asynchronously across 14 system events. Failed pushes retry 3× with exponential backoff.

→ Async · Resilient · 14 Events
💰

Commission Engine

Commission auto-calculated on every completed order at the restaurant's configured rate. Payout reports for vendors and admin.

→ Auto-Calculated · Configurable
🏪

Restaurant Approval Flow

Admin-controlled onboarding with approval/rejection workflow. Vendors receive push + SMS + email on outcome with a configurable reason string.

→ Governed · Auditable
🖥️

Super Admin Panel

Full platform oversight — all restaurants, users, orders, coupons, commissions, banners, and support tickets from one dashboard with Recharts analytics.

→ Full Oversight · Analytics
The Applications

THREE APPS.
one backbone.

All client apps talk to a single unified Node.js REST API — enforcing the same business rules, RBAC, and real-time event model regardless of interface.

📱 React Native
👤

Customer App

The primary consumer experience — a single React Native codebase for iOS and Android that makes ordering intuitive, fast, and reliable.

✓Restaurant discovery with cuisine filters & Google Maps address
✓Menu browsing with full variant and modifier support
✓Embedded Razorpay checkout — UPI, cards, wallets
✓Live order status push notifications at every step
✓Coupon redemption, order history, and post-delivery reviews
✓Offline-resilient via React Query stale-while-revalidate
📱 React Native
🛵

Delivery Agent App

A lightweight, focused task-management app built for speed — keeping agents informed and moving without unnecessary complexity.

✓Online / offline availability toggle with real-time backend sync
✓High-priority FCM/APNs task push that wakes the app
✓Google Maps / Apple Maps deep-link navigation
✓Full task lifecycle — Assigned → Picked Up → Delivered
✓Optional background GPS reporting for customer tracking
🌐 React SPA · Vite
🖥️

Vendor & Admin Panel

A single React SPA that renders the Restaurant or Admin Panel based on the JWT role claim — no duplicate codebase, no duplicate deployment.

✓Real-time order board and kitchen view via Socket.IO
✓Menu, table, staff, agent, and coupon management
✓Sales & commission analytics with Recharts dashboards
✓Admin: full platform oversight, approvals, and settings
✓Tablet-optimized views for waiters and kitchen staff
✓React Hook Form + Zod schema validation throughout

Why FoodBridge

DESIGNED FOR
real-world SCALE

🏗️

Modular by Design

Each service module is cleanly isolated with well-defined interfaces. Features can be developed, tested, and deployed independently — without disrupting the rest of the platform.

⚡

Real-Time First

Socket.IO WebSocket connections power every live experience — order boards, kitchen displays, and customer tracking all update instantly with zero polling overhead.

🔐

Security at Every Layer

RBAC enforced on every endpoint, HMAC-verified payment webhooks, JWT with short-lived access tokens, OTP-only authentication, and rate limiting at the gateway layer.

📱

Unified Codebase

One React Native codebase for iOS and Android. One React SPA for Vendor and Admin. One Node.js API for all clients. Zero duplication, maximum consistency.

☁️

Cloud-Native Infrastructure

Containerised with Docker, deployed via GitHub Actions CI/CD to AWS ECS Fargate. Auto-scaling, zero-downtime deploys, and Sentry + Grafana observability built in.

📊

Data-Driven Operations

Pre-aggregated analytics for every role — vendors track sales and commissions, admins monitor platform health, and everyone exports to CSV with one click.
